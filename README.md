# Visual Programming Course Lectures Content 🖥️

This repository contains the source code and practical examples covered during the **Visual Programming** course. The course focuses on building desktop applications using **C#** and **Windows Forms**.

## 📌 Course Syllabus & Covered Topics

The repository includes code examples for the following modules:

* **Introduction to Windows Forms**: Project structure, Form class, and Properties.
* **User Interface Design**:
    * Basic Controls: `Label`, `TextBox`, `Button`, `CheckBox`, `RadioButton`.
    * Containers: `Panel`, `GroupBox`, `TabControl`.
    * Lists: `ListBox`, `ComboBox`.
* **Event-Driven Programming**: Handling Click, Load, and other common events.
* **Navigation & Menus**: `MenuStrip`, `ToolStrip`, `StatusStrip`, and `ContextMenuStrip`.
* **Dialogs**: Working with `MessageBox`, `OpenFileDialog`, and `SaveFileDialog`.
* **Data Management**:
    * **File I/O**: Reading and writing text files.
    * **Database Connectivity**: Basics of **Entity Framework**.
    * **Data Presentation**: `DataGridView`, `DataBinding`, and CRUD operations (Displaying/Editing records).

## 🚀 How to use this repository
1. Clone the repository.
2. Open the solution file (.sln) using **Visual Studio**.
3. Navigate through the folders to find specific lecture examples.

## 🛠️ Requirements
* Visual Studio (2022 recommended).
* .NET Framework / .NET SDK.
* SQL Server (for database-related examples).
